# SEO Audit Toolkit

SEO Inspector is a browser extension designed to help developers, SEO specialists, and content teams quickly analyze the technical SEO structure of any webpage.

## Features

* Heading hierarchy analysis (H1–H6)
* Internal and external link validation
* Image SEO validation (`alt`, `title`, sizes)
* Metadata inspection
* DOM structure analysis
* Quick SEO reports
* Visual debugging tools
* URL-based execution rules

## Perfect for

* Technical SEO audits
* Frontend validation
* Content QA
* Accessibility reviews
* Performance-oriented SEO workflows

Analyze pages faster without opening multiple tools or external platforms.
